# 4-corners-maliks-adventures

Travel outside the gates of the kindom and see what wild lives in the desert! Play as Malik and slay creatures and level up!

## Controls

p - 'pause'

direction:

w - 'up'
s - 'down'
a - 'left'
d - 'right'

fight mechanics:

j - 'weapon attack'
k - 'magic attack'
h - 'weapon change'
n - 'magic change'
